# 🏢 CRM Module - نظام إدارة علاقات العملاء

## 📋 نظرة عامة

نظام CRM متكامل لإدارة العملاء والعملاء المحتملين والحملات التسويقية مع تكامل كامل مع Odoo وMeta Ads.

## 🗂️ هيكل المجلد

```
CRM/
├── pages/                    # صفحات CRM
│   ├── crm-advanced.tsx     # لوحة تحكم CRM الرئيسية
│   ├── customer-360.tsx     # عرض شامل للعميل
│   └── campaign-tracker.tsx # تتبع الحملات التسويقية
├── components/              # مكونات CRM
│   └── CRMDashboard.tsx    # مكون لوحة التحكم
├── services/               # خدمات API
│   └── crm.service.ts      # خدمة CRM الرئيسية
├── hooks/                  # React Hooks
│   └── useCRM.ts          # Hook للتعامل مع CRM
├── types/                  # تعريفات TypeScript
│   └── crm.types.ts       # أنواع البيانات
├── utils/                  # أدوات مساعدة
│   └── crm.utils.ts       # دوال مساعدة
├── styles/                 # ملفات التصميم
│   └── crm.css            # تصميم CRM
└── README.md              # هذا الملف
```

## 🚀 الميزات الرئيسية

### 👥 إدارة العملاء
- عرض قائمة العملاء مع التصفية والبحث
- إضافة وتعديل بيانات العملاء
- تتبع حالة العملاء (نشط، معلق، غير نشط)
- حساب قيمة العميل الإجمالية

### 🎯 إدارة العملاء المحتملين
- تتبع العملاء المحتملين حسب المراحل
- نظام تسجيل النقاط للعملاء المحتملين
- تحويل العملاء المحتملين إلى عملاء فعليين
- تتبع مصادر العملاء المحتملين

### 📊 تتبع الحملات
- مزامنة مع Meta Ads
- تتبع أداء الحملات (CPL, CTR, ROAS)
- تصدير البيانات إلى BigQuery
- تحليل ROI للحملات

### 🔍 عرض العميل 360
- دمج بيانات من Odoo وMeta وGoogle Analytics
- عرض شامل لتاريخ العميل
- تحليل سلوك العميل
- توصيات ذكية للمتابعة

## 🛠️ الاستخدام

### استيراد المكونات

```typescript
import CRMAdvancedPage from './CRM/pages/crm-advanced';
import { Customer360 } from './CRM/pages/customer-360';
import { CampaignTracker } from './CRM/pages/campaign-tracker';
import { useCRM } from './CRM/hooks/useCRM';
```

### استخدام Hook

```typescript
const {
  customers,
  leads,
  campaigns,
  loading,
  createCustomer,
  updateLeadStage,
  syncWithMeta
} = useCRM();
```

### استخدام الخدمات

```typescript
import { CRMService } from './CRM/services/crm.service';

const crmService = new CRMService();
const customers = await crmService.getCustomers();
```

## 🔧 التكامل

### Odoo CRM
- مزامنة العملاء والفرص
- تتبع الأنشطة والمهام
- إدارة خط المبيعات

### Meta Ads
- جلب بيانات الحملات
- تتبع تكلفة العميل المحتمل
- حساب ROAS

### Google Analytics
- تتبع سلوك العملاء
- تحليل مسار التحويل
- قياس فعالية الحملات

## 📈 المقاييس والتحليلات

- **إجمالي العملاء**: عدد العملاء الكلي
- **العملاء النشطين**: العملاء الذين تم التفاعل معهم مؤخراً
- **إجمالي القيمة**: مجموع قيم جميع العملاء
- **معدل التحويل**: نسبة تحويل العملاء المحتملين
- **متوسط قيمة الصفقة**: متوسط قيمة العملاء
- **تكلفة اكتساب العميل**: التكلفة لكل عميل جديد
